Javascript/jQuery UI Demos

  • Better Back to Top - keeps track of where you came from so you can go back from top
  • Help Overlays - a way to automatically provide overlays on anything that has a "help" attribute
  • Key Mapping - an attempt to duplicate Ableton Live's key mapping system
  • Fixed Scrolling - Had a problem with normal "autoscrolling" with "onscroll" in javascript so found a new method using CSS properties
  • Scroll Adjust - demo of forcing a div to autoscroll when new content is added only if already at the bottom
  • Dancing Bits - An interactive media piece I was playing with. It definitely doesn't work with IE, other browsers maybe. Definitely works in Chrome and Firefox 3.Only tested in Firefox 3.0.
  • Default Checkboxes - A way to handle "nothing selected" sensibly by assuming defaults (ie, no search criteria selected makes no sense, so use these search filters by default).
