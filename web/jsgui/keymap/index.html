Key Mapping Demo

Instructions:Simply press the "Setup Key Bindings button". Anything mapable will turn orange immediately. Then, select the UI element you want to map by clicking on it. Finally, press the button on the keyboard you want to use as a map (except f5 and esc which are reserved for browser refreshing and closing the mapping respectively.) Once you have it mapped, exit mapping by clicking the button again or pressing escape. Currently does not work in IE (only tested with 8)

It should be noted that the current solution is beyond hacky but shows promise and can certainly be made into a library if needed.
