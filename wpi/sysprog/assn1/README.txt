This program reads in some number of products (as the original does) and then prints them out in order.

ABOUT
-----
Score is conventionally used on a scale of 1-100 like grades.  But higher scores are treated as better.

Price on the other hand is better the cheaper it is.  0 is the best since it is free.

There is no insurrance against entering in negative numbers, but they would be viewed as worse than free, but better than everything else.

INSTALL
-------
simply untar and then run "make" in the current directory


RUN
---
