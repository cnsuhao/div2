This program tests various methods and creations of the Employee class and its derived classes: Manager and HighLevelManager.

ABOUT
-----
Each test is fairly well documented when run.  It should be noted that minimal manipulation on the created objects takes place and objects are sometimes reused.  Also, for the last test, 1 is true and 0 is false.

INSTALL
-------
simply untar and then run "make" in the current directory


RUN
---
To run, enter "./employee"
