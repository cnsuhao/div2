Program to test read and write file functions

INSTALL
-------
simply untar and then run "make" to get the first main program.  "make stest2" and "make stest3" to get the other two.


RUN
---
To run, enter "./stest", followed by "./stest2" to create "stest2.txt" and finally "stest3.txt" to read file created by stest2

CLEAN
---
run "make clean", does not delete text files.
