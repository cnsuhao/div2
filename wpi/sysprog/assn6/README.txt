A program to test a stack, queue, and tree sort struct

stacktest tests pushing and popping onto a limited stack
queuetest tests enqueueing and dequeueing on a limited queue
tnodetest tests a basic tree sort with a simple phrase
tnodetest2 tests the same sort with randomly generated chars

INSTALL
-------
untar, then run
make (stacktest)   //stack test is default
make queuetest
make tnodetest
make tnodetest2


RUN
---
run the programs previously created
./stacktest
./queuetest
./tnodetest
./tnodetest2

CLEAN
---
run "make clean"
